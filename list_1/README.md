# Lista 1 – Regressão Linear: OLS, GD e SGD

Disciplina: CKP8277 – Aprendizagem Automática · UFC

## Conteúdo

### Questão 1 — Regressão Linear Simples
Implementação dos três métodos de ajuste de regressão linear sobre um dataset de peixes (comprimento × peso):

- **OLS** (Ordinary Least Squares): solução analítica via equação normal
- **GD** (Gradient Descent): gradiente descendente em batch
- **SGD** (Stochastic Gradient Descent): gradiente descendente estocástico

### Questão 2 — Regressão Polinomial com Regularização L2
Regressão polinomial aplicada ao dataset **California Housing**:

- Divisão treino/teste (80%/20%)
- Treinamento sem regularização
- Treinamento com regularização L2 (λ = 0.01)
- Comparação de desempenho entre os modelos
